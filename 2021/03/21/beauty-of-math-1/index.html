数学之美之一—抽象与公理化

本文是一篇随笔

抽象是一种不太容易发现的美，用几个数学概念为例，谈一下抽象的美.

数学上的抽象往往是一个数学模型的建立，这个模型是根据具体问题提出来的. 我们发现诸多现实问题ABCD，它们都存在一些共同特征，于是我们从这些具体的问题中提取这些共同特征，构建一个抽象的模型$\pmb{K}$. 接着，我们研究模型$\pmb{K}$的性质. 这样任何具有这样特征的具体问题ABCD，便就都具有模型K的性质.

科学就是发现共性和差异性，把诸多的现实问题抽象和归类，这是共性（构建模型$\pmb{K}$）；再应用到不同的具体问题中，这是具体（ABCD具有$\pmb{K}$的性质）. 这样由具体到抽象，再由抽象到具体的美妙特征，在数学中能完美的体现.

数域$\pmb{K}$上所有$n$元有序数组组成的集合$\pmb{K^n}$，对于有序数组的加法和数量乘法运算，成为数域$\pmb{K}$上的一个线性空间.

数域$\pmb{K}$上所有$s\times n$矩阵组成的集合，对于矩阵的加法与数量乘法运算，成为数域$\pmb{K}$上的一个线性空间，记作$M_{S\times n}(K)$.

多项式、复数域、数域$\pmb{K}$本身都可以看成线性空间. 这些例子表明，线性空间这一数学模型的适用性很广. 如果可以从线性空间的定义出发（该模型的特点），研究线性空间的性质，它们将对于所有具体的线性空间都成立. 包括核磁中的$K$空间.

与抽象有关的另一个词是公理化. 所谓公理就是公认的道理，不需要证明，也不需要定义，它们做为问题讨论的前提出现，是问题探讨的起点.

我们举几个例子.

第一个是欧式几何，定义欧式几何是5条公理：

  1. 从一点向另一点可以引一条直线；
  2. 任意线段能无限延伸成一条直线；
  3. 给定任意线段，可以以其一个端点作为圆心，该线段作为半径作一个圆；
  4. 所有直角都相等；
  5. 若两条直线都与第三条直线相交，并且在同一边的内角之和小于两个直角，则这两条直线在这一边必定相交.

由此我们导出了丰富的几何系统（初高中所有几何知识均属于欧式几何）

第二个是概率，概率的公理化定义非常有意思，是前苏联大数学家柯尔莫哥夫实现的，称之为柯式公理体系. 在该体系中，引进一个定义在$\mathcal{F}$（可简单理解为事件的集合）上的函数$P$（即概率）. 对$\mathcal{F}$中任一成员$A$，$P(A)$解释为事件$A$的概率. 柯式公理体系对这个函数$P$加上了几条要求(即公理)：

  1. $1 \le P(A) \le 1, \text{ } \forall A \text{ in } \mathcal{F}$;
  2. $P(\Omega) = 1, P(\phi) = 0$;
  3. $\mathcal{F}$中的两互斥事件$A,B$，指 $P(A+B) = P(A)+P(B)$.

在这三个简单的公理上，建立起了概率论丰富的大厦.
